# Data Analytics Portfolio

This repository contains business-focused data analytics projects showcasing dashboarding, KPI analysis, and insight generation.

## Projects

### Sales Insights Dashboard – Power BI
Analyzed revenue, profit, and customer contribution across markets using Power BI and DAX.
### KPI Reporting Automation – Python & Excel
Automated weekly KPI reporting pipeline that validates raw data, aggregates business metrics, and generates standardized Excel reports.
### A/B Testing & Experiment Analysis – Python
Designed and analyzed controlled A/B experiments to evaluate conversion rate improvements using statistical hypothesis testing.
